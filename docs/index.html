Zup Open Source Tech Radar

Tech radar

The Radar is a document that sets out the changes that are currently interesting regarding software development in our organization.
It is a knowledge base about where we are today as an organization and where we want to go.

Aim of the tech radar

It centralizes things in motion that we should pay attention to and consider using in our projects.
The original concept comes from thoughtworks, you can read more about it here.

Potencial

These solutions may bring values. It is necessary to assess and test them in our environment to confirm or not this vision.

Deprecated/ Rejected

For various reasons, these solutions are no longer the first choices or are not recommended (anymore).
